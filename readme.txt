Show the next or previous post in a box that appears when the user scrolls to the bottom of a current post.

== Description ==

<a href="https://crestaproject.com/downloads/cresta-posts-box/" taget="_blank">Plugin Homepage & Demo</a>

With <strong>Cresta Posts Box</strong> you can show, in a single page (posts, pages or custom post types), a small box that allows the reader to <strong>go to the next or previous post</strong>. The box appears only when the reader finishes reading the current post.

<strong>Some features</strong>
<ul>
	<li>Choose to show next or previous post</li>
	<li>Choose the box position (left or right)</li>
	<li>Choose the font size</li>
	<li>Choose the box width</li>
	<li>and more...</li>
</ul>

<strong>PRO version features</strong>

== Installation ==

1. Unzip cresta-posts-box.zip
2. Upload the folder 'cresta-posts-box' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress Dashboard
4. Go to WordPress Main Menu -> CPB FREE to set the options


== Frequently Asked Questions ==

= Can I choose which pages to display the box? =
Yes, in the plugin options page you can choose which pages to display box (pages, posts, media and custom post types).

= Can I hide the box in a few pages? =
Yes, just edit the page/post you want to hide the box, find the metabox called "Cresta Posts Box" and check the box "Hide Cresta Posts Box in this page".

= Can I show the box on the blog feeds, category pages, tag pages, etc? =
No, the box can be shown only on the individual pages (pages, posts, media and custom post types).

= I activated the plugin but the box does not appear =
The box will automatically appear when the reader scrolls to the bottom of a current post.
If the problem persists, there may be a javascript error that stops the plugin execution, then check that your website does not have javascript errors.
